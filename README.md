# MyFilmStrip

Turn your photos into vintage film strips. Built with Next.js 14, React, Tailwind CSS, and canvas-based image processing.

## Features

- Upload up to 4 photos, or capture them with a 4-shot camera countdown
- Drag-and-drop to reorder, shuffle button to randomize
- 3 canvas-based film recipes: Classic B&W, Warm Vintage, Film Camera
- Realistic film strip output with perforations, borders, and film labels
- Download as PNG

## Local development

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Deploy

Push to GitHub, then connect the repo in Vercel.
